<h1> crazy-collisions </h1>

Well, well, well, this is the game I've been developing for the last two months.

Crazy Collisions is a roguelike, pretty silly, where you click the screen to generate balls and make them collide. After completing the round's objective before time runs out, you'll go to a shop with three slots filled with random and varied items.
